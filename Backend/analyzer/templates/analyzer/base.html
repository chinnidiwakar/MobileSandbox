Welcome to Mobile Sandbox

MobileSandbox is a tool to that takes a Android application (apk-file) and analyzes it for malicious behaviour. In order to analyze a sample please register an account or use the anonymous upload function. If you want to search for an already submitted sample you can use the search bar by entering the SHA1-, SHA256 or MD5 sum of the sample.

Please note: This service is run purely as a research tool and a best effort service. We reserve the right to take it down at any point for maintenance or other reasons.
